# Exclusions / Esclusioni

**[IT]** Pattern di esclusione configurabili tramite la chiave `Exclusions` di `config.ini`. I file il cui nome corrisponde a uno qualsiasi dei pattern non vengono sincronizzati in nessuna direzione.

**[EN]** Exclusion patterns configurable via the `Exclusions` key of `config.ini`. Files whose name matches any pattern are not synchronized in either direction.

---

## Syntax / Sintassi

```ini
Exclusions=*.csv|*.prm|*.xls
```

**[IT]** Lista di pattern glob separati dal carattere pipe `|`. Spazi attorno ai pattern vengono ignorati. La chiave e opzionale: assente o vuota -> nessuna esclusione.

**[EN]** A list of glob patterns separated by the pipe `|` character. Whitespace around patterns is trimmed. The key is optional: missing or empty -> no exclusions.

### Wildcards / Caratteri Jolly

| Symbol | Meaning / Significato |
|--------|------------------------|
| `*` | Zero or more characters / Zero o piu caratteri |
| `?` | Exactly one character / Esattamente un carattere |

**[IT]** Il matching e realizzato con `PathMatchSpecA` (libreria `shlwapi`): e *case-insensitive* e si applica al solo nome file, non all'intero path.

**[EN]** Matching is performed with `PathMatchSpecA` (`shlwapi` library): it is *case-insensitive* and applies to the file name only, not the full path.

---

## Examples / Esempi

### Office Temp Files / File Temporanei Office
```ini
Exclusions=~$*|*.tmp
```
**[IT]** Esclude i file temporanei generati da Word/Excel quando aprono un documento.

**[EN]** Excludes temporary files generated by Word/Excel when opening a document.

### Skip Spreadsheets and CSV / Salta Spreadsheet e CSV
```ini
Exclusions=*.csv|*.prm|*.xls|*.xlsx|*.xlsm
```

### Editor Backup Files / File di Backup di Editor
```ini
Exclusions=*.bak|*.swp|*~|*.orig
```

### Windows System Junk / Spazzatura di Sistema Windows
```ini
Exclusions=Thumbs.db|desktop.ini|*.lnk
```

### Combined Use Case / Caso d'Uso Combinato
```ini
Exclusions=*.tmp|~$*|Thumbs.db|desktop.ini|*.bak|*.log
```

## Behavior by Operation / Comportamento per Operazione

### Upload (WATCHER)

| Event / Evento | Action / Azione |
|----------------|-----------------|
| `FILE_ACTION_ADDED` su file escluso | Skip (logged as `[EXCLUDED] Upload skipped`) |
| `FILE_ACTION_MODIFIED` su file escluso | Skip (logged as `[EXCLUDED] Upload skipped`) |
| `FILE_ACTION_REMOVED` su file escluso | Skip (logged as `[EXCLUDED] Delete skipped`) |

### Rename (WATCHER)

| Old Name | New Name | Action / Azione |
|----------|----------|-----------------|
| Excluded / Escluso | Excluded / Escluso | Skip (logged as `[EXCLUDED] Rename skipped`) |
| Excluded / Escluso | Not excluded / Non escluso | Upload of new path / Upload del nuovo path |
| Not excluded / Non escluso | Excluded / Escluso | Delete of old path / Delete del vecchio path |
| Not excluded / Non escluso | Not excluded / Non escluso | Rename via RNFR/RNTO (with delete+upload fallback) |

### Download (POLLER)

**[IT]** Durante l'enumerazione del FTP, i file il cui nome corrisponde a un pattern di esclusione vengono saltati silenziosamente: niente download, niente entry in `g_recentlyProcessed`.

**[EN]** During FTP enumeration, files whose name matches an exclusion pattern are silently skipped: no download, no entry in `g_recentlyProcessed`.

## Use Cases / Casi d'Uso Tipici

### 1. Volatile Files / File Volatili
**[IT]** Documenti aperti in Office generano file `~$*.docx` che cambiano in continuazione: escluderli evita rumore inutile sulla rete.

**[EN]** Office documents being edited generate `~$*.docx` files that change constantly: excluding them avoids needless network noise.

### 2. Sensitive Data / Dati Sensibili
**[IT]** Estrazioni temporanee in formato CSV o XLS che non devono lasciare il client.

**[EN]** Temporary CSV or XLS extractions that must not leave the client.

### 3. One-Way Source of Truth / Sorgente di Verita Mono-Direzionale
**[IT]** Per garantire che certi tipi di file rimangano **solo** sul client, mettili in `Exclusions`: il POLLER ignorera eventuali copie remote.

**[EN]** To ensure certain file types remain **only** on the client, list them in `Exclusions`: POLLER will ignore any remote copies.

## Limitations / Limitazioni

**[IT]**
- I pattern operano sul nome del file, non sul percorso. `temp/*.txt` non funziona; usa `*.txt` (e tutti i `.txt` saranno esclusi).
- Le directory non possono essere escluse: i pattern si applicano ai file. Per saltare un'intera cartella considera di sincronizzare un parent diverso.
- I pattern vengono ricaricati solo al riavvio del servizio.

**[EN]**
- Patterns operate on the file name, not on the path. `temp/*.txt` does not work; use `*.txt` (and all `.txt` files will be excluded).
- Directories cannot be excluded: patterns apply to files. To skip an entire folder consider syncing a different parent.
- Patterns are reloaded only on service restart.
